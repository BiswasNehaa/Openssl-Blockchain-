
AES(Advanced Encryption Standard) encryption => symmetric encryption

1. echo "Hello_World" > b.txt       # b.txt is the file name with Hello text inside it

2. openssl enc -aes-256-cbc -in b.txt -out b.bin
 give password for encryption


3. openssl enc -base64 -in b.bin -out b.b64     # It converts binary file → Base64 text


4. openssl enc -d -aes-256-cbc -in b.bin -out b.dec  #for decryption




Generate RSA (Rivest – Shamir – Adleman) Private key:  (Asymmetric private key)

1. openssl genkey -algorithm RSA -out prt.pem -pkeyopt rsa_kepygen_bits:1024     #generate private key
2. openssl pkey -in prt.pem -pubout -out pub.pem                                 #extracts the PUBLIC KEY from your PRIVATE KEY
3. openssl pkeyutl -encrypt -pubin -inkey pub.pem -in b.txt -out rsa_encrypted.bin   #encrypt using public key
4. openssl pkeyutl -decrypt -inkey prt.pem -in rsa_encrypted.bin -out rsa_decrypted.txt  #decrypt using private key
